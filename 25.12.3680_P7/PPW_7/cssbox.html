Contoh CSS Box Model

Ini adalah contoh implementasi CSS Box Model yang terdiri dari margin, border, padding, dan content. Paragraf ini juga menggunakan tipografi dari Google Fonts, dengan pengaturan line-height agar lebih mudah dibaca, serta text-align justify untuk tampilan teks yang rapi di kedua sisi.

Ilustrasi: Diagram CSS Box Model (Margin > Border > Padding > Content)

Properti Fungsi
Margin Ruang di luar border (transparan)
Border Garis batas yang mengelilingi padding dan konten
Padding Ruang antara border dan konten (transparan)
Content Area tempat teks dan gambar ditampilkan
