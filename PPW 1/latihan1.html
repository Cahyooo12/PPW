Deskripsi Buku

HTML5 merupakan pengembangan dari versi HTML sebelumnya. Web Hypertext Application Technology Working Group (WHATWG) mulai membuat standar baru ini pada tahun 2004 ketika W3C sedang fokus pada pengembangan XHTML 2.0 di masa depan, sementara HTML 4.01 belum pernah diperbarui sejak tahun 2000. Sejak tahun 2009,W3C dan WHATWG bekerja sama dalam pengembangan HTML5. Meskipun HTML5 telah dikenal luas oleh para pengembang web sejak lama, HTML5 baru mencuat pada April 2022 April 2010. Untuk membuat teks rata tengah pada HTML5, gunakan kode CSS berikut:text-align:center;.
