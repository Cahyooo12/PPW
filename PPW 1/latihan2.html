Jenis-Jenis Link

1. Link Relatif: Link ke halaman dokumen HTML yang lain. Contoh: Klik link berikut
2. Link Absolut: Link ke internet. Contoh: www.amikom.ac.id
3. Link Mail to: Link ke email yang dituju. Contoh: Kirim email ke Yayasan@amikom.ac.id
4. Link ke bagian lain dalam dokumen. Link ini masih terdapat dalam satu dokumen atau file yang sama. Caranya yaitu dengan klik link berikut: (perhatikan URL sebelum dan sesudah diklik.)

Bagian Lain Dokumen HTML

Link atau hyperlink adalah elemen HTML yang berfungsi menghubungkan suatu halaman web ke halaman web yang lain. Dari sisi pengguna, elemen ini bisa diklik. Setelah diklik, maka pengguna akan diarahkan ke halaman lain sesuai alamat URL yang diberikan.
